API Wizard

지도 Web API의 주요 기능에 대한 코드를 자동으로 생성해주는 기능입니다. 생성된 코드를 직접 수정해서 확인하고 싶을 경우에는 '직접해보기'를 이용해 보세요. 보다 상세한 기능은 Sample과 Documentation을 참고하세요.
(지도나 로드뷰를 이동/확대/축소하면 왼쪽의 관련 옵션값도 함께 변경됩니다. 마커, 커스텀오버레이, 도형의 위치는 화면에서 바로 확인 가능하도록 지도 중심에 위치시키고 있습니다.)
